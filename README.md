# 🇬🇧 매일 영어 3문장

매일 영어로 3문장을 쓰면 **AI가 첨삭**해 주고, **진척도를 관리**하며,
표준발음·연음발음을 **소리로 들을 수 있는** 영어 학습 웹앱입니다.

서버가 필요 없는 **단일 HTML 파일**이라, 파일 하나만 열면 바로 사용할 수 있어요.

## 주요 기능

- **매일 3문장 영작** — 오늘의 주제를 보고 영어 문장 3개를 작성
- **AI 첨삭** — 문장별로 자연스러운 원어민 표현으로 교정 + 한국어 설명
  - 무엇이 왜 틀렸는지(전치사, 시제, 어휘 등) 한국어로 알려줌
  - 문장별 0~100점 점수
- **발음 듣기**
  - 🔈 **표준발음** — 또렷하고 천천히 (단어 구분이 잘 들림)
  - 🎧 **연음발음** — 자연스러운 속도로 연결해서 (실제 대화 속도)
  - 🔗 **연음 가이드** — 단어가 어떻게 연결되는지 한글 힌트 제공 (예: `pick it up → 피키럽`)
- **진척도 관리**
  - 🔥 연속 학습일(streak), 누적 문장 수, 학습한 날, 평균 점수
  - 일별 평균 점수 추이 그래프
  - 지난 기록 다시 보기 (발음 재생 포함)
- **목표 & 리마인더** — 하루 목표 문장 수 설정, 진행 막대, 리마인더 시간에 브라우저 알림
- **주간 리포트** — 최근 7일 학습일/문장수/평균 점수, **전주 대비 변화**, 자주 틀린 표현 Top 5, 코멘트
- **복습 노트** — 틀린 표현을 자동으로 모아 복습. **퀴즈 모드(정답 가리기)**, 복습 완료 체크, 반복 횟수, 발음 재생
- **API 키 없이 공유** — 프록시 서버(`proxy/`)를 배포하면 사용자는 키 없이 사용 (아래 참고)

모든 학습 기록은 **내 브라우저에만** 저장됩니다(localStorage).

앱은 상단 탭으로 나뉩니다: **✍️ 오늘 / 📈 진척도 / 📊 주간 리포트 / 📒 복습 노트**.

## 사용 방법

### 1. 앱 열기
`index.html` 파일을 웹 브라우저(Chrome, Edge, Safari 등)로 열면 됩니다.

### 2. AI 첨삭 연결 (둘 중 하나)
AI 첨삭 기능은 Anthropic(Claude) API를 사용합니다. **⚙️ 설정 → 연결 방식**에서 고르세요.

- **직접 API 키 입력 (개인용, 기본값)**
  1. [console.anthropic.com/settings/keys](https://console.anthropic.com/settings/keys) 에서 API 키 발급
  2. 설정에 붙여넣고 저장 → 키는 **이 브라우저에만** 저장되고 Claude API를 직접 호출
- **프록시 서버 사용 (키 숨김·공유용)**
  - `proxy/` 폴더의 서버를 배포하고 그 주소를 설정에 입력
  - 사용자는 API 키를 몰라도 됩니다. 자세한 방법은 [`proxy/README.md`](proxy/README.md)

### 3. 매일 학습하기
1. 오늘의 주제를 보고 문장 3개를 영어로 작성 (1~2개만 써도 됩니다)
2. **AI 첨삭받기** 클릭
3. 교정 결과·설명·점수를 확인하고, 🔈 표준발음 / 🎧 연음발음으로 들어보기

### 발음만 사용하기
발음 재생은 브라우저 내장 음성(Web Speech API)을 쓰므로 **API 키 없이도** 동작합니다.
음성 종류는 설정에서 바꿀 수 있어요.

### 목표 · 리마인더 · 리포트 · 복습
- **목표/리마인더**: 설정에서 하루 목표 문장 수와 리마인더 시간을 정하고, `🔔 브라우저 알림 켜기`를 누르면
  앱을 열어둔 상태에서 그 시간에 알림이 옵니다. (목표 미달 시 상단에 배너로도 안내)
- **📊 주간 리포트**: 최근 7일 요약과 전주 대비 변화, 자주 틀린 표현 Top 5를 자동 집계합니다.
- **📒 복습 노트**: 첨삭에서 나온 틀린 표현이 자동으로 쌓입니다. `정답 가리기`로 퀴즈처럼 연습하고,
  외운 표현은 `완료`로 체크하세요. 탭의 빨간 배지는 복습 대기 개수입니다.

## 온라인에 올리기 (선택)

GitHub Pages로 무료 배포할 수 있습니다.

1. 이 저장소의 **Settings → Pages**
2. Source를 배포 브랜치로 지정
3. 발급된 주소로 접속 (예: `https://<사용자명>.github.io/English/`)

## 폴더 구성

```
index.html      # 앱 전체 (단일 파일)
proxy/          # (선택) API 키를 숨기는 프록시 서버
  worker.js       - Cloudflare Workers 용 (권장·무료)
  wrangler.toml   - Cloudflare 배포 설정
  server.js       - Node 로컬/자체 호스팅 용
  README.md       - 프록시 배포 방법
```

## 기술 메모

- 단일 `index.html` (외부 라이브러리 없음)
- 저장: `localStorage` (학습 기록·복습 상태·설정)
- 발음: 브라우저 `SpeechSynthesis` (표준=느린 속도, 연음=자연 속도)
- 알림: 브라우저 `Notification` API (앱을 열어둔 동안)
- 첨삭: Claude Messages API — 직접 호출 또는 프록시 경유
- 그래프: 순수 SVG (외부 차트 라이브러리 없음)
- 다크 모드 자동 지원
